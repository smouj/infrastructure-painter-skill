# Infrastructure Painter

[![ES](https://img.shields.io/badge/ES-Español-red)](README.es.md)

Habilidad Infrastructure Painter para OpenClaw.

##Qué Soluciona
Genera automáticamente diagramas visuales de infraestructura en la nube.

## Características
- Activación automática cuando se detectan tareas relevantes
- Resultados profesionales listos para producción
- Enfoque de seguridad primero
- Soporte de reversión

## Uso
/infrastructure-painter

## Ejemplo
```
Analiza el código para identificar problemas de infraestructura.
```

## Instalación
Esta habilidad está disponible automáticamente en OpenClaw.
